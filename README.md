# Connecta
### Projeto de Estrutura de Dados Para Gestão de Eventos e Rede Social
  O Connecta foi um projeto feito na disciplina de Estrutura de Dados e apresentado no evento Acelera 2025 na Fatec Cruzeiro, com a proposta de ser um **sistema de gestão de eventos**, que permite ao usuário a criação, inscrição e gestão de eventos presenciais ou online. O projeto também conta com funcionalidades de **rede social**, como um feed de postagens, que possibilita a criação de publicações de texto, imagens e vídeos, e interação com comentários e curtidas.
  O projeto foi feito sem a utilização de banco de dados, foram utilizadas estruturas de dados.
